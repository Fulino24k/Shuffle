# Reflection/Diary App


## Requirements
- Node.js
- Web server

## Setup Instructions

1. Install Node.js on your system:

2. Make sure Node.js works by:
    ```bash
     node --version
     ```

3. Install dependencies:

    ```bash
    npm install
    ```

4. Run the app:
    ```bash
    npm run dev
    ```

5. Access the app:
   - Open your browser and navigate to:
     - Local web server: `http://localhost:****`
